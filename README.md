# 🚀 FastPick
> **AI 기반 이미지 선별 및 자동 분류 솔루션** > 수천 장의 사진 속에서 베스트 샷을 찾고, 유사한 이미지끼리 알아서 묶어주는 스마트한 도구입니다.

[![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)

---

## ✨ Key Features
* **Smart Selection**: AI 알고리즘을 통해 흔들림, 구도, 표정을 분석하여 최적의 사진을 추천합니다.
* **Auto Clustering**: 유사한 구도의 사진들을 그룹화하여 중복 선택을 방지합니다.
* **Fast Preview**: 고해상도 이미지도 끊김 없이 빠르게 렌더링하여 검토 시간을 단축합니다.
* **Batch Export**: 분류된 결과물을 원하는 폴더 구조로 한 번에 내보낼 수 있습니다.
